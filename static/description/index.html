User Export Module for YODY

Export thông tin users dễ dàng và nhanh chóng

Tính năng chính

  • Export users theo nhiều tiêu chí khác nhau
  • Hỗ trợ định dạng Excel và CSV
  • Lựa chọn các trường thông tin cần export
  • Lưu lịch sử các lần export
  • Phân quyền người dùng rõ ràng

Cách sử dụng

  1. Vào menu User Export > Export Users
  2. Chọn loại export và các trường cần xuất
  3. Click Export để tạo file
  4. Download file về máy
